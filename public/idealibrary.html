COMIC HUB

Hosting space for web comic artist in the northwest that connects web comic artist together. Olympia/Seattle as a starting base. The idea is that anyone could potentially start there own website deticated to comics. Each website would be linked to a larger website that gathered data that could be arranged in interesting ways including most active page most viewed page, etc. It could also work as an RSS feed for web comics already existing. A social networking site just for comics. Or maybe just an aggregator.

Columns Columns Columns!!!

Let's be honest columns could be really really cool on this site. How those columns get organized could be even cooler. My idea is columns would have cute little graphics or illustrations that would show up next to the title when they get posted to the main page. A column page could have the graphic, the name of the column, and a brief description of the writer and content of the column. There would also be a reverse chronlogical listing of all the previous entries. The more specific the columns the better. Example column titles/content could be: The Cats on my block,  gardening adventures: the evergrowing story, Sundays: Cooking rice with Charlie D, Opinions and experiences in enviormental preservation, 24 hours: stories from timefighters. I really like the last one because it promotes a column where everyone could contribute.

I could go on forever but thats all for now.
